# Yansi-WMS-Vue3

一个基于 **Vue 3 + Vite** 开发的仓库管理系统前端项目，用于实现物资管理、库存管理、出入库记录、统计分析等功能。

## 技术栈
- Vue 3
- Vite
- JavaScript
- Element Plus

## 功能模块
- 登录与退出
- 统计概览
- 物资管理
- 库存管理
- 出入库管理
- 个人中心
- 系统设置

## 项目运行

```bash
npm install
npm run dev
```

## 项目打包

```bash
npm run build
```

## 项目说明
当前项目为前端部分，部分功能使用 Mock 数据模拟，后续可对接真实后端接口。
