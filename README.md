# Yansi-WMS-Vue3
这是一个基于 Vue 3 + Vite 构建的项目模板，用于快速开始开发。

## 推荐 IDE 配置
[VS Code](https://code.visualstudio.com/) + [Vue 官方插件 Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)  
（请禁用旧插件 Vetur）

## 推荐浏览器配置
- Chrome / Edge / Brave 等 Chromium 内核浏览器：
  - 安装 [Vue.js 开发者工具](https://chromewebstore.google.com/detail/vuejs-devtools/nhdogjmejiglipccpnnnanhbledajbpd)
  - 在开发者工具中开启「自定义对象格式化器」

- Firefox 浏览器：
  - 安装 [Vue.js 开发者工具](https://addons.mozilla.org/en-US/firefox/addon/vue-js-devtools/)
  - 在开发者工具中开启「自定义对象格式化器」

## 自定义配置
参考官方文档：[Vite 配置文档](https://vite.dev/config/)

## 项目命令

### 安装依赖
```sh
npm install
```

### 开发环境启动（热更新）
```sh
npm run dev
```

### 生产环境打包
```sh
npm run build
```
